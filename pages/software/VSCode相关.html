# 插件推荐

在这里搜索和安装插件

# Atom One Light Theme

护眼绿(豆绿色主题设置)

Ctrl+Shift+P搜索配置文件

把如下代码粘贴进去(如果你有别的, 那可以把它补充进去)

{
    "workbench.colorTheme": "Atom One Light",
    "workbench.colorCustomizations": {
        "[Atom One Light]": {
            "editor.background": "#C7EDCC",   
            "sideBar.background": "#C7EDCC",
            "activityBar.background": "#C7EDCC",       
        },
    },
}
