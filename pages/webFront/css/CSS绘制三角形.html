# 拉伸三角形

其实就是设置边框的大小, 上下小一点,左边是上下边框的4倍,右侧none 

width: 0;
height: 0;
border-left: 200px solid red;
border-top: 50px solid transparent;
border-bottom: 50px solid transparent;
border-right: none;
